# Disease Detector
this program is an ML app to disease detection from exists symptoms.

# Install
```bash
sudo apt install virtualenv python3-pip
virtualenv -v venv
soruce venv/bin/activate
pip install -r requirements.txt
```

# Run
```bash
python3 app.py
```
